CSS Cascading Style Sheet

CSS is the languags used to describe the presentation of HTML and to style the web pages like colors,size ,spacing,layout,animations,accessibility hints ,ect

PWD full form is:-responsive web design:it is the approach of resolutions and device capabilities.

3 way of styles to insert to html

1.inline styling

2.internal styling

3.external styling

.........1.inline styling.........in css we written directly in the HTML elements using the style attribute

..........2.internal styling ........it is used to write the code inside the style tag in the HTML files head section.

..........3.external styling.........css kept in seperate .css file and it is linked to the HTML using link tag inside the head tag.
